- [시연 영상 보기](https://youtu.be/9dNp8FtCwFk) 👀

## 구현기술

- 기술스택 : Java, Html, Css, Javascript, PHP
- 웹서버 : Apache
- 데이터베이스 : MySQL
- 운영체제 : Ubuntu, Android OS
- 프로토콜 : HTTP, TCP/IP, SFTP, SSH
- API/라이브러리 : Volly, HttpURLConnection, IBM Watson, Gson
